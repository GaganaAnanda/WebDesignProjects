# Amma's Kitchen - South Indian Recipes Website

## Description
A simple website showcasing South Indian recipes with images, videos, audio guides, and healthy diet tips. Includes forms for subscription and contact.

## HTML Tags Used
- `<!DOCTYPE html>` : Defines HTML5 document
- `<html>` : Root element
- `<head>` : Metadata and links to CSS
- `<meta>` : Charset and viewport info
- `<title>` : Page title
- `<link>` : CSS and favicon
- `<body>` : Main content container
- `<header>` : Top section with navigation
- `<nav>` : Navigation bar
- `<ul>` / `<li>` / `<a>` : Lists and links
- `<main>` : Main content area
- `<section>` : Defines sections of content
- `<article>` : Self-contained content or recipes
- `<h1>` / `<h2>` / `<h3>` : Headings of different levels
- `<p>` : Paragraphs of text
- `<table>` / `<tr>` / `<th>` / `<td>` : Tables for ingredients
- `<figure>` / `<img>` / `<figcaption>` : Images with captions
- `<iframe>` : Embeds YouTube videos
- `<audio>` / `<source>` : Audio guides
- `<details>` / `<summary>` : Expandable content (diet tips)
- `<form>` / `<input>` / `<label>` / `<datalist>` / `<option>` / `<button>` : Subscription form elements
- `<aside>` : Sidebar content (quick links and contact info)
- `<footer>` : Footer section with copyright and social links
