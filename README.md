#Web Design Assignments

This repository contains my coursework and practice assignments for web design and front end development. Each assignment demonstrates different concepts including HTML structure, CSS styling, responsive layouts, interactive forms, JavaScript features, media integration, React components, and full stack basics using React and Node.
