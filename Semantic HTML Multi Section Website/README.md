# Gagana Ananda - Professional Portfolio

A responsive professional portfolio website built with HTML5 and CSS3.

## HTML Tags Used

### Document Structure
- `<!DOCTYPE html>` - Declares HTML5 document type
- `<html>` - Root element of HTML document
- `<head>` - Contains metadata about the document
- `<body>` - Contains visible page content

### Metadata Tags
- `<meta charset="UTF-8">` - Sets character encoding to UTF-8
- `<meta name="viewport">` - Controls responsive design on mobile devices
- `<title>` - Sets browser tab title
- `<link rel="icon">` - Adds favicon to browser tab
- `<link rel="stylesheet">` - Links external CSS file

### Semantic Structure Tags
- `<header>` - Contains site navigation and branding
- `<main>` - Wraps primary content of the page
- `<section>` - Defines thematic sections of content
- `<article>` - Contains standalone content within sections
- `<aside>` - Contains supplementary content (stats section)
- `<footer>` - Contains site footer information
- `<nav>` - Wraps navigation links

### Content Organization Tags
- `<div>` - Generic container for grouping elements
- `<h1>, <h2>, <h3>, <h4>` - Heading hierarchy for content structure
- `<p>` - Paragraph text content
- `<ul>` - Unordered list for navigation and footer links
- `<li>` - List items within unordered lists

### Navigation Tags
- `<a>` - Anchor links for navigation and external links

### Table Tags
- `<table>` - Creates data table for skills section
- `<thead>` - Groups table header content
- `<tbody>` - Groups table body content
- `<tr>` - Table rows
- `<th>` - Table header cells
- `<td>` - Table data cells

### Form Tags
- `<form>` - Contact form wrapper
- `<label>` - Form field labels for accessibility
- `<input>` - Text and email input fields
- `<textarea>` - Multi-line text input
- `<button>` - Submit button and action buttons

### Media Tags
- `<img>` - Images for projects and testimonial avatars
- `<figure>` - Wraps project images and captions
- `<figcaption>` - Project descriptions

## CSS Selectors Used

### Element Selectors
- `*` - Universal selector for CSS reset
- `body` - Global body styling
- `header` - Header section styling
- `footer` - Footer section styling

### Class Selectors
- `.container` - Main content wrapper
- `.hero` - Hero section styling
- `.content-card` - Content section backgrounds
- `.btn` - Button base styles
- `.btn-primary` - Primary button styling
- `.btn-secondary` - Secondary button styling
- `.nav-menu` - Navigation menu styling
- `.experience-list` - Experience section container
- `.experience-item` - Individual experience card
- `.company-logo` - Company initials/logo container
- `.skills-table` - Table styling
- `.gallery` - Project gallery container
- `.project` - Individual project cards
- `.testimonials-container` - Testimonials section wrapper
- `.testimonial-card` - Individual testimonial card
- `.contact-form` - Contact form styling
- `.form-group` - Form input grouping
- `.stat` - Stats section layout
- `.relationship` - Testimonial relationship description

### ID Selectors
- `#about` - About section anchor
- `#experience` - Experience section anchor
- `#skills` - Skills section anchor
- `#projects` - Projects section anchor
- `#testimonials` - Testimonials section anchor
- `#contact` - Contact section anchor

### Pseudo-classes
- `:hover` - Hover effects for interactive elements
- `:focus` - Focus states for form inputs

### Responsive Selectors
- `@media (max-width: 768px)` - Tablet responsive styles
- `@media (max-width: 375px)` - Mobile responsive styles
